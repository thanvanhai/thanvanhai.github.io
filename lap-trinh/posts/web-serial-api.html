Web Serial API cho dự án IoT

Ngày đăng: 27/05/2026 • Tác giả: Hải Còi • Danh mục: Lập trình

Web Serial API cho phép web app giao tiếp trực tiếp với thiết bị qua cổng serial trên trình duyệt hỗ trợ chuẩn mới.

Ví dụ kết nối

const port = await navigator.serial.requestPort();
await port.open({ baudRate: 9600 });

Bạn có thể áp dụng để đọc dữ liệu cảm biến hoặc điều khiển module ngoại vi.

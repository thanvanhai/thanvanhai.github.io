## Về tác giả

Xin chào, mình là **Hải Còi** — lập trình viên với nhiều năm kinh nghiệm phát triển phần mềm và triển khai hệ thống ERP doanh nghiệp.

Blog này là nơi mình ghi lại và chia sẻ những kiến thức, kinh nghiệm thực tế trong quá trình làm việc, bao gồm:

## Nội dung chính

### 🖥️ Lập trình

- **Java** — Spring Boot, microservices, design patterns
- **C#** — .NET, LINQ, Entity Framework
- **PHP** — Laravel, API development
- **AutoIT** — automation scripts, Windows tools
- **JavaScript / TypeScript** — Web APIs, Node.js
- **Python** — scripting, data processing

### 🗄️ Database

- **SQL Server** — tuning, indexing, query optimization
- **Oracle** — PL/SQL, partitioning, EBS integration
- **MongoDB** — aggregation pipeline, schema design

### 🏭 ERP Nghiệp vụ

- **Oracle EBS** — Manufacturing, Inventory, Costing
- **Epicor** — Production, Materials, BOM/Routing
- **SAP / Microsoft Dynamics** — kiến trúc so sánh

### [PATC - Catthai Corp](https://catthaicorp.com/)
**05/2021 — 11/2023**

Làm chủ toàn diện hệ sinh thái **Epicor ERP** — từ tầng dữ liệu đến tùy biến nghiệp vụ:

- **Database** — SQL Server, BAQ (Business Activity Query), thiết kế và tối ưu truy vấn trên schema Epicor
- **BPM (Business Process Management)** — viết Method Directive, Data Directive để can thiệp luồng nghiệp vụ theo yêu cầu đặc thù
- **Customization** — tùy biến giao diện (Customization/Personalization), thêm trường dữ liệu (UD Fields), tạo Dashboard, BAQ Report
- **Tích hợp** — REST API, kết nối Epicor với hệ thống bên ngoài
- **Phân hệ nghiệp vụ** — triển khai đầy đủ Production, Materials Management, BOM/Routing, Inventory, Costing

### [PSC - Pyramid Software and Consulting](https://psctelecom.com.vn/)
**05/2015 — 03/2021**

Nắm vững toàn diện từ **code, nghiệp vụ đến database**, phát triển giải pháp ERP trên nền **C#**, chuyên sâu về **phân hệ Tài chính** cho các đơn vị công lập và một số khách hàng ngoài công lập, với các công nghệ:

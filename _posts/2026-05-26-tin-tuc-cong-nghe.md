## Tổng hợp tuần này

Tổng hợp các xu hướng, framework và công cụ mới đáng chú ý dành cho lập trình viên trong tuần qua.

### 🚀 JavaScript / Web

- **Bun 1.x** tiếp tục cải thiện tốc độ build và bundling so với Node.js
- **Next.js App Router** ngày càng được cộng đồng React adopt rộng hơn
- **Web Serial / WebUSB / WebBluetooth** API phổ biến dần trong dự án IoT

### ☕ Java / JVM

- **Java 21 LTS** — Virtual Threads (Project Loom) chính thức, giúp viết async code đơn giản hơn nhiều
- **Spring Boot 3.x** native image với GraalVM giảm thời gian khởi động xuống dưới 100ms

### 🔷 .NET / C\#

- **C# 12** primary constructors giảm boilerplate đáng kể
- **Minimal API** trong ASP.NET Core tiếp tục là lựa chọn nhẹ cho microservice

### 🗄️ Database

- **PostgreSQL 16** cải thiện logical replication và query parallelism
- **MongoDB 7.x** Atlas Search được tích hợp sâu hơn với vector search cho AI

### 🏭 ERP & Enterprise

- Xu hướng "cloud ERP" tăng mạnh — SAP S/4HANA Cloud và Oracle Fusion đang được nhiều doanh nghiệp vừa và lớn xem xét migration
- Low-code integration platform (Boomi, MuleSoft) giúp giảm custom code trong tích hợp ERP
